# 02-领域知识清单

## 1. 业务规则清单
1. 用户可输入纯路径或 `METHOD + 路径`。
2. 路由匹配需支持“类级 + 方法级”路径拼接。
3. 多命中不能直接报错，需要用户选择。
4. 未命中需要有明确提示。

来源：PRD《3.1 F1》

## 2. 验收口径清单
1. `/user/add` 能命中 `add`。
2. `/user/update` 能命中 `update`。
3. 常见 Spring 注解命中率 >= 90%。

来源：PRD《5.验收标准》

## 3. 技术边界清单
1. 当前版本聚焦 URL 跳转。
2. Services Tree、HTTP 工具、JSON 工具不在本期。
3. JAX-RS 延期；Kotlin 为可选。

来源：PRD《3.2 本期不做》《7.版本计划》
