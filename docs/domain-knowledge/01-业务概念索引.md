# 01-业务概念索引

## 1. Endpoint
定义：可被 URL 命中的接口方法定义。

来源：PRD《1.产品目标》《5.验收标准》

## 2. 类级路径
定义：声明在 Controller 类上的请求路径，例如 `/user`。

来源：PRD《3.1 F1》（类级路径 + 方法级路径）

## 3. 方法级路径
定义：声明在方法上的请求路径，例如 `/add`、`/update`。

来源：PRD《5.验收标准》

## 4. 完整路径
定义：类级路径与方法级路径拼接后的路径，例如 `/user/add`。

来源：PRD《3.1 F1》《5.验收标准》

## 5. 命中结果
定义：URL 匹配后的结果类型：唯一命中、多命中、未命中。

来源：PRD《3.1 F1 命中逻辑》
