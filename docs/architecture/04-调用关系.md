# 04-调用关系

## 1. 主调用链
`gotoByUrlCommand` -> `endpointIndexer` -> `urlMatcher` -> (`quickPick`) -> `methodNavigator`

来源：PRD《4.交互说明》《3.1 F1 命中逻辑》

## 2. 时序说明
1. 命令触发后先保证索引可用。
2. 匹配完成后根据结果分支处理。
3. 最终由导航器执行编辑器跳转。

来源：PRD《4.交互说明》

## 3. 依赖方向约束
1. `command` 可依赖 `indexer/matcher/navigator`。
2. `matcher` 不得依赖 `vscode UI`。
3. `navigator` 不得反向依赖解析模块。

来源：PRD《6.非功能要求》（可维护性）
